Cayenne Documentation - v. 3.0

Cayenne documentation consists of the following "books":

  • Overview: Flavors of Cayenne; what's new in the latest version.
  • Cayenne Guide: Writing applications based on Cayenne persistence API. Includes "getting started" tutorials, main concepts, API details and deployment procedures.
  • Remote Object Persistence Guide: Writing distributed Cayenne client applications.
  • Modeler Guide: A guide to object-relational mapping (ORM) with CayenneModeler GUI tool.
