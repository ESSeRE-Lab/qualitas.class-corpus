1. Introduction

Look and Feel plugin using the Liquid style from http://liquidlnf.sourceforge.net/.
This was simply repackaged for Columba as plugin.


2. Installation
Select Utilities->Plugin Manager to open the Plugin Manager Dialog. Select the
"Install..." button which opens a dialog where you have to choose the location
of the zip-file containing your plugin.

3. Usage
Select Edit->General Options to open the dialog, which lets you select the new L&F.
