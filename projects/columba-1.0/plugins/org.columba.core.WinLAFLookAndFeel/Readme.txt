============================ WinLAF Look and Feel ===========================

1. Introduction
2. Installation
3. Usage
4. Author


1. Introduction

Look and Feel plugin using the Windows emulation shipped with the JDK. Additionally, it 
uses the WinLAF patch package from http://winlaf.dev.java.net/, to fix issues, which
aren't fixed in JDK until now.


2. Installation
Select Utilities->Plugin Manager to open the Plugin Manager Dialog. Select the
"Install..." button which opens a dialog where you have to choose the location
of the zip-file containing your plugin.

3. Usage
Select Edit->General Options to open the dialog, which lets you select the new L&F.
