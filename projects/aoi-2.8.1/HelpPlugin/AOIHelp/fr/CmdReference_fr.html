Référence des Commandes
(Extrait du manuel écrit par Julian MacDonald et traduit par Les Manuels d'Aide en Français)



1 Raccourcis Clavier

Pour accélérer votre débit de travail, plusieurs outils et fonctions possèdent un raccourci clavier. Ils sont résumés ci-dessous:



Fonctions Fichier:

Ctrl+N - Créer un nouveau fichier AoI
Ctrl+O - Ouvrir un fichier AoI existant
Ctrl+W - Fermer le fichier AoI en cours
Ctrl+S - Enregistrer le fichier AoI sous le même nom
Ctrl+Q - Quitter Art of Illusion


Fonctions Édition:

Crtl+Z - Annuler/Refaire la dernière action
Ctrl+X - Couper les objets sélectionnés vers le presse-papiers
Ctrl+C - Copier les objets sélectionnés dans le presse-papiers
Ctrl+V - Coller les objets dans un fichier à partir du presse-papiers
Ctrl+A - Sélectionner tous les objets de la scène
Suppr - Effacer les objets sélectionnés


Fonctions Objet:

Ctrl+E - Éditer un Objet
Ctrl+L - Éditer l'agencement d'un objet
Ctrl+T - Ouvrir la boîte de dialogue Transformer un Objet
Ctrl+U - Régler la Texture des objets sélectionnés
Ctrl+M - Régler le Matériau des objets sélectionnés

Les touches de direction sont utilisables pour déplacer/pivoter les objets sélectionnés dans le plan de la fenêtre sélectionnée si les icônes Déplacer/pivoter sont actives. Maintenir Ctrl enfoncée en appuyant sur les touches haut/bas déplace/pivote selon les autres axes. Maintenir Alt enfoncée en appuyant sur les touches de direction déplace/pivote les objets par 10 pixels.


Fonctions Animation:

Ctrl+P - Aperçu de l'Animation
Ctrl+] - Avancer d'une image
Ctrl+[ - Reculer d'une image
Ctrl+J - Atteindre le point temps ...
Ctrl+D - Éditer l'image clé sélectionnée
Ctrl+K - Piste(s) sélectionnée(s) de l'image clé
Ctrl+Maj+K - Pistes modifiées de l'image clé
Ctrl+Maj+A - Sélectionner toutes les pistes des objets sélectionnés


Fonctions Scène:

Ctrl+R - Ouvre la boîte de dialogue du Convertisseur
Ctrl+Maj+R - Convertit immédiatement avec les réglages en cours
Ctrl+B - Bascule entre un mode d'affichage et quatre modes d'affichage
Ctrl+G - Ouvre la boîte de dialogue Grille
Ctrl+F - Sélectionne une image avec l'appareil photo
Ctrl+Maj+F - Cadre la scène avec l'appareil photo
Ctrl+Maj+U - Ouvre la boîte de dialogue Textures
Ctrl+Maj+M - Ouvre la boîte de dialogue Matériaux


Fonctions Éditeur de Maillage:

Ctrl+Z - Annuler/refaire la dernière action
Ctrl+A - Sélectionner tous les sommets/bords/faces
Ctrl+X - Sélection étendue
Ctrl+F - Basculer en mode de sélection main libre
Ctrl+W - Affichage quadrangulaire
Ctrl+M - Ouvrir la boîte de dialogue Tension du Maillage
Ctrl+E - Éditer les points sélectionnés
Ctrl+T - Transformer les points sélectionnés
Ctrl+B - Biseauter/Extruder la sélection
Ctrl+P - Ouvrir la boîte de dialogue Paramètres de Texture
Ctrl+S - Régler le lissage des sommets/bords sélectionnés
Ctrl+R - Convertir l'aperçu
Ctrl+D - Ouvrir la boîte de dialogue Editer l'Os
Ctrl+G - Ouvre la boîte de dialogue Grille

Les touches de direction sont utilisables pour déplacer les points sélectionnés d'un pixel à la fois sur le plan de l'affichage si l'icône Déplacer est active. Maintenir Ctrl enfoncée en cliquant sur les touches haut/bas déplace le point selon les autres axes. Maintenir Alt enfoncée en cliquent sur les touches de direction déplace les points par 10 pixels.



Vous pouvez aussi définir d'autres raccourcis clavier via Edition->Préférences->Raccourcis.
