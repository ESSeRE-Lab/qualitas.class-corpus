The HowTo section needs content!
The concept is for a set of small tutelets that explain how to do some of the more popular tasks that new users ask about. To keep the total HelpPlugin download within reason, each tutelet would typically have no more than 3 screen-shots, and 10 paragraphs of text.
Useful tutelets could include:
- How to setup good default lighting
- How to render a Quicktime movie
- How to group objects using the null object
- How to setup good indoor lighting
- How to setup good outdoor GI lighting
- How to setup an environment dome
- How to use ambient occlusion
- How to use caustics to generate GI

I realise that any of this requires time. I may try generating some of this myself, but I thought that perhaps some of you might have ideas for some of these, based on the questions you've asked/answered on the forums, and/or content you have lying around from test scenes, and/or tutorials, etc.

Feel free to post anything regarding Help content to Nik777 on any of the forums.
