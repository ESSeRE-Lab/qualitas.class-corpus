(Contenu parrainé...)

Cette section a besoin de contenu !

Le concept est d'écrire un jeu de petits tutoriels qui montrent comment réaliser les taches les plus courantes auxquelles se livrent les nouveaux utilisateurs. Pour conserver une taille raisonnable au greffon d'aide en ligne, chaque tutoriel ne devrait pas dépasser trois copies d'écran et dix paragraphes de texte.
La liste des sujets traités pourrait être :
- Comment réaliser bon un éclairage de base
- Comment rendre une animation Quicktime
- Comment grouper les objets en utilisant l'objet null
- Comment créer un éclairage d'intérieur
- Comment créer un éclairage d'extérieur avec l'illumination globale
- Comment réaliser un dôme d'environnement
- Comment utiliser l'occlusion ambiante
- Comment générer une illumination globale avec les caustiques

Je réalise que chacun de ces tutoriels demande du temps. J'essairai peut être d'en écrire moi-même, mais peut être certain d'entre vous ont des idées à ce sujet, tirées des discussions survenues dans les forums.

N'hésitez pas à m'envoyer tout ce qui peut avoir trait à l'aide en ligne à Nik777 sur n'importe lequel des forums AoI.
